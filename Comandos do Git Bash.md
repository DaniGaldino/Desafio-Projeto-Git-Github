# Comandos GitBash 

- **cd** - *Navegar por diretórios*

- **ls** - *listar arquivos*

- **mkdir** - *criar pasta*

- **rm-rf** - *deletar tudo de forma forçada*

- **cd .. ** - *retroceder*

- **Ctrl+l** - *limpar*

- **mv** - *mover arquivo*

- **git add .** - *adicionar tudo ao stage*

- **git status** - *mostra status do diretório*

- **git commit -m "comentário"** - *cria um commit atual*
